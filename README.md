## ℹ️ What is MechDesignCopilot?

MechDesignCopilot helps you move from design requirements to a complete bill of materials (BOM) quickly. It simplifies complex mechanical and mechatronic designs. You do not need engineering experience or specialized software knowledge to use it.

This app uses AI to assist with automation, documentation, and manufacturing tasks. It supports CAD and CAE workflows and helps you organize your mechanical and robotics projects.

Topics include automation, design templates, and integration with AI tools to speed up your workflow. MechDesignCopilot works on Windows for end users with no coding required.

## 🛠 Troubleshooting Common Issues

- **Installer won’t start**  
  Make sure you have Windows 10 or later and run the installer as Administrator. Right-click the file and select **Run as administrator**.

- **App won’t open after installation**  
  Restart your computer. Check you meet system requirements. Make sure your antivirus isn’t blocking the app.

- **AI features do not respond**  
  Confirm you have an active internet connection. AI features rely on online services.

- **Project files won’t save or load**  
  Check if you have write permissions in the folder where you save files. Try saving to the Desktop or Documents.
